# 🛡️ CTF Writeups – Protocol Zero

Welcome to my CTF Writeups repository.

This repository is part of **Protocol Zero**, my long-term cybersecurity learning journey. Here I document the challenges I solve, the techniques I learn, the mistakes I make, and the lessons I take away from each CTF.

My goal isn't just to collect flags—it's to build practical skills in digital forensics, web security, reverse engineering, cryptography, and incident response while creating resources that are useful for other learners.

## 🎯 Learning Philosophy

Every challenge teaches something new.

Sometimes you'll solve a challenge independently.

Sometimes you'll need hints.

Sometimes you'll follow a write-up to understand a new technique.

All three are part of learning.

If you're just starting out, don't wait until you feel "ready." Start today, stay curious, and keep building one challenge at a time.
